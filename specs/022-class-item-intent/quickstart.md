# Quickstart: Class-Item Intent Lock

## V1 — Classic unchanged

1. Build with classic exotic armor hash A.
2. PATCH to classic hash B without identityAction → 409.
3. confirm/fork → succeeds.

## V2 — Intent swaps

1. Build exotic armor = exotic class item (or unset + class_item exotic on variant).
2. Variant A class_item X+perks; Variant B class_item Y+perks.
3. Save both without identity confirm; coverage may show soft tiers.

## V3 — Mode flip

1. Classic exotic → class item (or reverse) without identityAction → 409.
2. confirm → mode updates.

## V4 — Debug

1. Builds debug: set exotic armor; observe mode; resolve/coverage; Sets debug for perks.
